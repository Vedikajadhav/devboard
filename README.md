# 📋 DevBoard — Task Manager

A full stack task management app built with React, Node.js, and MongoDB.

## 🚀 Live Demo
Coming soon...

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Database:** MongoDB Atlas

## ✨ Features
- ✅ Add new tasks
- ✅ Mark tasks complete/incomplete
- ✅ Delete tasks
- ✅ Data saved in cloud database
